ly_crawler
==========

由立法院網站取得立法院議事錄，並產生純文字格式檔案。

Extract links of journals from Taiwanese Congress website.
Data come from http://lci.ly.gov.tw/

執行程式需求
------------
* Python 2.7+
* python module: lxml

安裝
----
* 建議使用Linux
* 下載程式 https://github.com/jyunfan/ly_crawler/zipball/master
* 解壓縮至任意目錄
* 進入bin目錄
* 執行./process.sh
